揪IN（JOUIN）

不是外送平台，而是對傳統外送模式的修正。

傳統外送平台同時讓三方不滿意：

店家被高抽成壓榨

消費者花更多、吃更少、等更久

平台被迫維持高成本物流體系

揪IN 從一開始就選擇不走那條路。

傳統外送平台的結構性問題
對消費者來說

1️⃣ 餐點變少、價格變高

店家為了支付 25%～35% 抽成

縮水份量、提高單價

消費者實際買到的價值下降

2️⃣ 額外費用不斷疊加

外送費

平台服務費

高峰加價

小額訂單費

一餐還沒吃到，費用先疊好幾層。

3️⃣ 送餐時間不可預期

顯示 30 分鐘，實際 60 分鐘

餐點先做好但卡在外送員

到手已冷，體驗失敗

4️⃣ 被迫購買「即時配送」
多數人其實不需要立刻送到，只需要：

價格合理

份量正常

時間可預期

但外送平台只提供一種選項。

對店家來說

高抽成導致利潤被壓縮

不參與平台就沒有流量

平台掌握價格與顧客關係

揪IN 的核心差異
1️⃣ 不做物流，不養外送員

平台完全不處理配送

店家可選擇自送或自取

不將物流成本轉嫁給消費者

結果是：
平台結構更輕，費用更低。

2️⃣ 不靠抽成，靠訂閱

傳統外送：每單抽成

揪IN：月費 / 功能訂閱制

訂單越多：

店家越賺

消費者越划算

平台與雙方利益一致

3️⃣ 團購成團制，提高訂單密度

不追求一單一送

以團購集中需求

一次出餐，多筆成交

特別適合：

社區

辦公室

熟客圈

4️⃣ 先確定需求，再出餐

設定截止時間

達標才成團

出餐與取餐時間事前明確

不是賭外送運氣，
而是事前就講清楚。

5️⃣ 平台是「成交放大器」，不是中間商

平台只專注於：

成團機制

訂單與狀態追蹤

訂閱與金流管理

不干涉：

店家定價

出餐方式

顧客關係

一句話總結

外送平台賣的是「即時」，
揪IN 解決的是「合理」。

把零散、被迫、昂貴的消費，
變成可預期、可負擔、可持續的成交。
